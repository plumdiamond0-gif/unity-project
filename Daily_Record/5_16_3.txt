1. 로딩 시간 감소

씬 시작:
로비 입장

미리 넣은 경우:
로비 + 상점 UI + 보스 + 전투 이펙트
전부 준비

Addressables:
로비만 로드
↓
상점 열기 → 상점 UI 로드
보스전 시작 → 보스 로드

초기 로딩이 


2. 메모리 절약
예:
몬스터 100종
씬에 전부 연결:
100개 관련 텍스처/애니메이션

실제 게임:
이번 스테이지는 5종만 사용
Addressables:
Load("Slime");
Load("Goblin");
Load("Dragon");



3. 의존성 감소
씬이 가벼워짐.
미리 연결:
Scene
 └─ UI
    └─ Button
       └─ Icon
          └─ Texture
하나가 꼬이면 연쇄적으로 따라옴.

Addressables:
Scene
 └─ 필요할 때 가져옴

씬 자체가 훨씬 단순해짐.
