플레이어 생성 이전에 UI패널이 생성된다면?

해결->

등록 함수:
GameManager
public void RegisterPlayer(GameObject player)
{
    _player = player;
    TryBind();
}

UIManager
public void RegisterHUD(PanelPlayer hud)
{
    _hud = hud;
    TryBind();
}

RegisterPlayer, RegisterHUD 씬배틀에서 두개 로드할 때 호출

private void TryBind()
{
    if (_player == null || _hud == null)
        return;

    _hud.Bind(_player);
}

Player loaded or HUD loaded
→ RegisterPlayer orRegisterHUD
→ 둘 중 하나 없음 → Bind 안함
→ 둘 다 있음 → Bind
